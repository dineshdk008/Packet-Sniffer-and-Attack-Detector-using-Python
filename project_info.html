Project Information - Packet Sniffer

Project Overview

This project is a network packet sniffer designed to monitor and analyze network traffic. It captures packets, extracts relevant information, and sends alerts for sensitive data such as passwords and emails.

Key Features

  • Network Interface Selection: Allows users to select the network adapter for packet sniffing.
  • Packet Capture: Utilizes Scapy to capture live network packets.
  • Data Extraction: Extracts sensitive information like passwords and email addresses from captured packets.
  • Email Alerts: Sends email notifications when sensitive data is detected.
  • Threading: Uses threading to handle packet sniffing and email sending concurrently.

File Descriptions

File Name Description
main_sniffer.py Main application file that sets up the GUI, handles user input, and starts/stops packet sniffing.
sniffer_core.py Core functionality for packet sniffing, including packet processing and data extraction.
notifier.py Handles email notifications using SMTP when sensitive data is detected.

Usage Instructions

  1. Install required libraries: Scapy, Tkinter (usually included with Python), and any other dependencies.
  2. Run the main_sniffer.py script to launch the GUI.
  3. Select the desired network adapter from the dropdown menu.
